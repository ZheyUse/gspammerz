# v1.0.2

## Highlights
- Smart Auto Address system with country profiles, cascading region/city/dependent fields, ZIP previews, and searchable dropdowns
- Auto country detection from form fields or detected location data
- Smart survey detection for gender/sex, age, consent, email, phone, birthdate, school, course, year level, occupation, religion, household size
- Weighted submission modes: Plan (pre-calculated distributions) and Dice (per-response rolls)
- Slider-based weighting UI with randomize-weights action
- Packaged name lists with auto-load support for auto naming

## UI/UX
- New weight mode selector cards and randomize weights icon button
- Auto Address configuration modal and responsive layout improvements
- Stop button styling and better pointer/disabled states

## Data + Platform
- Address data hydration via country, geo, and postal APIs
- New test forms generator for smart detection validation
- Manifest updates: new host permissions, name list resources, version bump to 1.0.2

## Fixes
- Weighted pick logic now handles zero or negative weights safely
- Auto name generation uses exact list entries (no partial truncation)
- Removed noisy debug logging
