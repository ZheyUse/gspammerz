# v1.0.3

## Highlights
- Enhanced Smart Name Detection - now detects "Name of Participant", "Complete name", "Name of...", and other variations
- Smart Date Detection - detects native Google Forms date pickers and fills with today's date
- Smart Profession/Work Detection - detects "Occupation", "Profession", "Job", "Current profession/work" fields with 720+ profession database
- Smart Course/Strand Detection - now uses 441+ course list from markdown file, supports both SHS strands and college courses

## Smart Detection
- Extracted name patterns: firstNames (1,144 Filipino names), lastNames (355 surnames)
- Extracted profession list: 721 jobs including Doctor, Engineer, Teacher, etc.
- Extracted course list: 441+ options including STEM, HUMSS, ABM, GAS, TVL for SHS and BS courses for college
- Date detection now checks `question.type === 'date'` for native Google Forms date pickers
- Detection uses markdown files loaded via `web_accessible_resources` for reliable access

## UI/UX
- Extension warning badge - shows red warning badge when extension field is detected but disabled
- Updated logo fonts - "Spammer" uses **Play** font, "Z" uses **Tektur** font
- New icon SVGs for crisp rendering at all sizes (16x16, 48x48, 128x128)
- Icon exporter tool (`export-icons.html`) for converting SVGs to PNGs
- Removed Refresh Preview button - form preview auto-refreshes
- Removed footer progress bar from sandbox preview
