# v1.0.4

## Highlights
- **Auto Nationality Smart Detection** - automatically detects and fills nationality, citizenship, ethnicity, and ancestry fields
- **Email Collection Detection** - detects forms with "Collect email addresses" enabled and warns users to prevent submission failures

## Auto Nationality Smart Detection

- Detects nationality-related fields: "Nationality", "Citizenship", "Country of origin", "What is your nationality", etc.
- Detects ethnicity fields: "Ethnicity", "Ethnic origin", "Tribe", "Race"
- Detects ancestry fields: "Ancestry", "Ancestral origin", "Country of origin"
- Pool-based generation with 6 regional options: All Countries, Filipino, Asian, European, American, African, Oceanian
- 200+ nationalities in the database with common variations (ese, ian, ish suffixes)
- Smart local matching - optionally matches nationality based on detected address country
- Works with both text inputs and dropdown/choice questions

## Email Collection Detection

- Automatically detects when form has "Collect email addresses" enabled
- Shows blocking modal on form load with clear instructions
- Blocks "Start Submitting" button when email collection is active
- Displays warning banner during submissions if failures occur
- Explains fix: form owner must disable in Settings → General or use "Responder Input"

## Smart Detection - New Patterns

Extends existing detection with nationality-related field patterns:
- `/\b(nationality|citizenship|citizen|country\s+of\s+(birth|citizenship|origin)|what\s+is\s+your\s+(nationality|citizenship))\b/i`
- `/\b(ethnic\s*(origin|ity)|ethnicity|tribe|race)\b/i`
- `/\b(ancestry|ancestral|ancestors|origin\s*(country)?)\b/i`

## UI/UX

- New "Auto Nationality" section in General Settings with toggle, status badge, and configure button
- Nationality configuration modal with pool selector, specific nationality dropdown, and "Match Local" toggle
- Email collection modal with warning icon, detailed instructions, and link to Google support
- Consistent dark neon styling with `--spammerz-error` color for warnings
- Floating warning banner in bottom-right during failed submissions

## Bug Fixes

- Improved email collection detection using multiple methods: FB_PUBLIC_LOAD_DATA_ settings, DOM inspection, URL parameters, and page script scanning

## Code Additions

- `detectNationalityQuestions()` - detects nationality-related fields from form questions
- `createDefaultNationalityConfig()` - initializes nationality configuration
- `generateNationality()` - generates nationality based on config and address country
- `detectEmailCollectionEnabled()` - multi method detection for email collection feature
- `showEmailCollectionModal()` - displays blocking modal with fix instructions
- `showEmailCollectionWarning()` - shows floating warning during submissions
